Title: NFT Aggregator

Installation: install VSCode or any other compiler for python install flask and psychopg2 install postgress and open nft_db

Usage: with the usage of our database, allow people to find the information about NFT, and if there is no information in the db, it will search it in the Solana web service

Example:

input:2FE7HdBq9F8LtUsoq8r2hpEAGa2Uz6Dd5XoKscewFg8y
